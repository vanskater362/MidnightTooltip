# MidnightTooltip

MidnightTooltip is a World of Warcraft addon designed to enhance the tooltip functionality within the game. This addon provides customizable tooltips that improve the user experience by displaying additional information and features.

## Features

- Enhanced tooltip display with customizable options.
- Localization support for English (US) and German (DE).
- Easy integration with existing World of Warcraft UI.

## Installation

1. Download the MidnightTooltip addon from the official repository.
2. Extract the contents of the downloaded file into your World of Warcraft `Interface/AddOns` directory.
3. Ensure that the folder structure is maintained as follows:
   ```
   Interface/AddOns/MidnightTooltip/
   ```
4. Launch World of Warcraft and enable the addon from the AddOns menu on the character selection screen.

## Usage

Once installed, the MidnightTooltip addon will automatically enhance your tooltips. You can customize the settings through the in-game options menu, which can be accessed by typing `/midnighttooltip` or `/mtt` in the chat.
